How to use this script :

You need :
* ruby >= 1.9, ruby >= 2.0
* gem bundler

If you get the gem from this repo, do a:
```shell
gem install ./job_break-0.0.1.gem
```
and restart your shell.

to use it just type:
```shell
job_break
```
and restart your shell.

for more help
```shell
job_break -h
```
